Inspetor de Elementos

O inspetor de elementos no navegador possibilita que o usuário tenha contato com os códigos HTML e CSS da página e altere-os temporariamente.

Estrutura Básica do HTML

A estrutura básica do HTML conta com a tag head, onde está situado todo o cabeçalho, e a tag body, onde está situado o corpo do texto, a ser acessado pelo usuário.

(Voltar)

Tags

Tags são palavras escritas entre "<" e ">" que informam ao navegador a estrutura e o significado do conteúdo inserido em uma página Web. Temos tags textuais, que organizam o conteúdo da página e tags semânticas que organizam a estrutura de conteúdo de uma página.

(Voltar)

Atributos

Servem para que algumas características sejam adicionadas a um elemento, a uma tag. São compostas por um nome e um valor. Dois atributos importantes são os atributos id (identificador) e class (define a classe).

(Voltar)

Listas

São usadas para menus, por exemplo. HTML fornece suporte para 3 tipos de listas; ordenadas, não ordenadas e de definição. Ordenadas servem para listar dados com a necessidade de representar a sua ordenação de forma numérica ou alfabética. Não ordenadas quando não apresenta necessidade de ordenar. De definição são utilizadas quando precisamos listar itens e atribuirmos uma descrição a eles.
